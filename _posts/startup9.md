#헬스케어

>1 헬스케어 산업분야 소개

* 치료 부분 의료 서비스 + 질병 예방 및 관리
* 헬스케어 : 건강한 삶을 위한 모든 제품과 서비스 
* 힐세리온 : 스마트폰 + 휴대용 초음파기기
* 네오팩트: 게임과 재활훈련
* 재노플랜 : 뱉는 침 개인 유전자 분석
* 닥터 키친: 호텔셰프의 당뇨환자를 위한 레시피
* 지형도 : 데이터 측정 --> 데이터 통합 --> 데이터 해석 --> 건강증진 활동
** 데이터 측정 : 아이폰 watch, fitbit, misfit, 23andme
** 데이터 통합(전략적) : data platform(애플,구글) vs emr/ehr(개인건강관련 정보의 주체/삼성) 개인 vs 의료진
** 데이터 해석 : 앤더슨 암센터 (2013 ibm인공지능 왓슨의 활용연구, 미래에는 의사대체)/ peek :백내장 검사/ 의료서비스 어려운 지역에 보급 (o2o)
** 건강증진 활동 : 디바이스, o2o(online to offline), 3d 프린팅(개인을 위한 보청기, 귀모양에 맞춤/ 딜라이트), 상담 컨설팅(소울링: 모바일 이용한 비대면 상담)/ 네오팩트, 와이 브래인,zocdoc(o2o)


>2 헬스케어 산업발전을 향한 2가지 관점

* EMR(전자의료기록)/ EHR(전자건강기록)/ PHR(개인건강기록)
* 진단 정보 소유의 주체 : 의사의 진료과정/ 감기진단(의사의 배타적 진단)/ 측정된 데이터는 누구것? EMR(전자의료기록)/ EHR(전자건강기록): 의학 // 개인의 것이라는 건 PHR
* EHR: 전기 차트, PHR: 모바일 플랫폼/ 클라우드 서비스 
* 데이터를 통합하는 주체가 누가될것인가? EHR중심의 혁신 어려움 표준화된 전자차트의 부재, 고가의 장비, 전문인력 /PHR 중심의 혁신 한계점: 질환치료(질환이라고 판단, 치료하는 행위는 의사의 배타적 권리, 병원과 의사들의 동의와 협력이 필요, 자의적으로 질환을 판단/방심하면 안됨)

>3 현실화된 의료 혁신 사례

* 애플의 리서치킷 : 2015년 3월 오픈소프트웨어 리서치킷 발표 : 질환관련 빅데이터 수집 도구 // 2016년 3월 케어킷 발표
* Mpower앱 : 1만명 참가자 / 파킨슨병 판단 / 운동전후로 진단 테스트 : 파킨슨병을 악화시키는 데이터 수집
* 얼굴인식 알고리즘을 통해 아이의 감정분석 --> 자폐증 발견나이를 5세에서 2세로 줄임
* 개인유전자 분석 : 23andme 2006년 "앤워지스키" 설립
* 인터넷 주문 ,분석키트 발송, 침발송, 결과 받아봄, 의사의 관여없이 이용할수있음 / 서비스 판매 금지 명령
* Pfizer,존슨엔존슨과 공동연구 /해외 시장 진출/ 안젤리나 졸리: 유전자 검사를 통해 유방암발병확률이 높다는 이유로 정상적인 유방을 절제함
* 제노 플랜: 헬스뷰티 유전자분석 스타트업/ 핵심역량을 이용한 새로운 서비스/ 헬스뷰티 / Bm: 다이어트 모델 --> 차별화 어려움. 유전자분석으로 차별화 개인에게 맞는 다이어트/ 헬스뷰티는 자유로움/ 유전정보 데이터 빠른 확보, 향후 맞춤 의료 시장 진입에 유리, 아시아인 유전자 db구축

>4 헬스케어 스타트업이 고려해야 될 사항

* 의료혁신의 4가지 목표 : 예측 의료, 예방의료, 맞춤 의료, 참여 의료
* Leroy Hood: 4p에서 환자의 역할이 중요
* 애플 --> 데이터 통합 phr(참여의료)
* ibm watson --> 데이터 분석 / phr중심의 맞춤형
* 시장 포지셔닝, 4p 지향성,확장가능성
* 전세계적 고령화 빈부격차 : 건강 보험 재정 문제 / 소득관계없이 의료서비스 이용/ 민감한 재정 문제 ! 의료비 감축 프로세스 --> 의료 접근성 향상 
* 선도적 헬스케어 스타트업 : 민간 보험사 역할이 중요 --> 운동량 측적하면 보험료 낮춰줄게!
* 의료비 낮추는 가장 효과적인 방법 :건강관리/ 질병예방 : 예방과 진단이 점차 더 성장한다
* 고려사항 : 서비스 이용대상의 명확화 (의료기관 종사자인지 일반인지)
* 23andme: 의사를 통한 서비스가 아니었다. 개인유전정보 선도 --> 사업의 성장
* 의료관련 법적 자문의 필요성(의료법, 약사법 등등)
* 누구에게 팔섯인가, 어떠한 기능과 가치를 담을 것인가, 광고표현을 어떻게 할것인가/ 과대광고 오남용 금지/ 식약처 특정광고 문안
* 사토 카츠아키 : 모두가 린스타트업 수용/경쟁력이없음/ 스타트업 폐기론--> 가치중심.전략적 거점 선점
* 일본 장기불황 속 생필품 저가 판매 "다이소" 성공 / 사회구조 변화속에서 사회가치를 제공하는 스타트업이 성공

>5 헬스케어 산업 패러다임 변화

* 18~20c 초 : 공중 보건(전염병 예방및 확산 방지)
* 20c초~말 : 질병 치료
* 21c 이후: 건강한 삶(질병 예방 및 관리)
* 2050년 65세이상인구 비중 26퍼센트(선진국) 14퍼센트(신흥국) 로예상 / 노인성 만성질환 의료비 급증
* 의료비 지출 증가 : 소득 증가, 고령화, wellness증가로 의료비 지출 증가 추세, 한국은 gdp비중이 아직 낮으나, 향후 높을것
* IT와의 융합 : 의료 +ICT /모바일 기기 이용한 / 웨어러블 디바이스/ 정신과 진료 : 모바일로 상담


>6 헬스케어 시장
